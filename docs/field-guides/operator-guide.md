# EventFlow Operator Quick Guide

**Print and distribute at pre-event briefing.**

---

## Joining the Event

1. Open your phone browser → go to: **[your EventFlow URL]/op**
2. Enter your phone number (the one registered with the organizer)
3. Enter the event code: **____________**
4. Tap **Join Event**
5. Optional: tap "Add to Home Screen" to install as an app

---

## Reporting an Incident

**3 taps. No typing required.**

1. Tap the big red **REPORT INCIDENT** button
2. Select what is happening: Congestion / Medical / Security / Resource / Infrastructure / Other
3. Select how serious: Low / Medium / High / Emergency
4. Tap **SUBMIT REPORT**

You will see "Report received ✓" within 2 seconds.

---

## If You Have No Internet

If the app shows **OFFLINE** at the top:

- You can still submit reports — they will be saved and sent automatically when connectivity returns
- The button will say **SAVE OFFLINE** instead of **SUBMIT REPORT**

**SMS fallback** (if the app isn't working at all):

Send a text message to: **____________**

Format: `EF ZONE CATEGORY SEVERITY`

Examples:
- `EF GATE-N CONGESTION HIGH`
- `EF STAGE MEDICAL EMERGENCY`
- `EF FOOD RESOURCE LOW`

## Severity Guide

| Level | When to use |
|---|---|
| **Low** | Minor issue, informational only |
| **Medium** | Needs attention soon |
| **High** | Needs immediate response |
| **Emergency** | Urgent threat to safety |

---

## If You're Not Sure

Report it. It is always better to report something that turns out to be minor than to stay silent about something serious.
